Scripting Guide ✕

1. Global Variables

You can define global variables and bind them to any object's properties dynamically. Create a script and write:
var runeRadius = 150;
var myPadding = 10; Then, in the Inspector for any shape, enter runeRadius into the X, Y, or Scale/Radius text fields instead of a regular number. (You can also type / in any field to autocomplete your variables!)

2. Procedural Generation (ArcanistAPI)

Scripting does not replace the node-based UI, but acts as an advanced secondary system to procedurally generate shapes through code using the ArcanistAPI.

ArcanistAPI.createMagicCircle(name, parentId, props)

Generates a Magic Circle. Returns its newly generated ID string.

ArcanistAPI.createPolygon(name, parentId, props)

Generates a True Polygon node.

ArcanistAPI.createText(name, text, parentId, props)

Generates Arcane Text along an arc. The text param expects a string.

ArcanistAPI.clearAll()

Wipes the entire workspace hierarchy clean.

3. Example Script

Copy and paste this snippet into a new Script and Run it: 

ArcanistAPI.clearAll();

// 1. Create a base circle container
let rootId = ArcanistAPI.createMagicCircle("Base Layer", null, {
    radius: 200, 
    symmetry: 6, 
    doubleRim: true,
    rotationSpeed: 0.2
});

// 2. Add an inscribed star inside the base
ArcanistAPI.createPolygon("Inner Star", rootId, {
    radius: 190, 
    symmetry: 6, 
    connectPoints: true,
    connectPointsSkip: 2,
    symbolsAtPoints: true,
    symbolText: "☿♀♁♂♃♄",
    rotationSpeed: -0.1
});
// 3. Add curved text to the base
ArcanistAPI.createText("Rune Label", "IGNIS NATURA RENOVATUR", rootId, {
    radius: 215,
    fontSize: 14,
    letterSpacing: 8
});
